# Node.js - 전역객체(Global)

- fileLocation : __dirname, __filename 정보 출력과 리소스 경로 얻기
- console : 정보 출력하기
- consoleTime : 실행 시간 측정
- consoleCount: 횟수 측정
- timer, timer_async_interval : 지연 실행, 반복 실행
- process : 프로세스 정보
- processArg : 애플리케이션 실행 파라미터 활용하기
- processEvent : 프로세스 이벤트

- require : require() 함수 정보 출력
